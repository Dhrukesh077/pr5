<h2>📌 Project Overview</h2>

<p>
The <strong>Employee Management System</strong> is a menu-driven Python application developed using 
<strong>Object-Oriented Programming principles</strong>.

It demonstrates:
</p>

<ul>
<li>🔐 Encapsulation (Private Attributes)</li>
<li>🧬 Inheritance</li>
<li>🔁 Method Overriding</li>
<li>🧠 Match-Case Control Flow (Python 3.10+)</li>
<li>📂 Class-Based Structure</li>
</ul>

This project is ideal for showcasing strong OOP fundamentals in Python.
</p>

<hr>

<h2>🏗️ System Architecture</h2>

<h3>👤 Base Class: Employee</h3>

<ul>
<li>Private attributes: ID, Name, Age, Salary</li>
<li>Method to display employee details</li>
</ul>

<h3>👔 Derived Class: Manager</h3>

<ul>
<li>Inherits from Employee</li>
<li>Adds Department attribute</li>
<li>Extends display functionality</li>
</ul>

<h3>💻 Derived Class: Developer</h3>

<ul>
<li>Inherits from Employee</li>
<li>Adds Programming Language attribute</li>
<li>Extends display functionality</li>
</ul>
